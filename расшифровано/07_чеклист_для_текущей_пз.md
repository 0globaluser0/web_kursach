## Что обязательно упомянуть по сайту

- Сайт находится в `site/`.
- Сайт работает без сервера, через HTML/CSS/JavaScript.
- Данные сохраняются в `localStorage`.
- Есть авторизация и регистрация.
- Есть демо-трейдер и демо-администратор.
- Администратор видит операции всех пользователей.
- Пользователь видит только свой портфель и свою историю.
- Есть темная тема.
- Есть логическая модель из семи связанных сущностей.
